# Classe Docfam {#core-ref:63fa0a57-efe6-4bf1-853d-0caa3b2f88f0}

La classe `DocFam` est la classe utilisée pour les [familles][famintro] de
documents. Tous les documents "famille" héritent de la classe `DocFam`.
